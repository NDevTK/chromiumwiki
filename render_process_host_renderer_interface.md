# RenderProcessHost and the Renderer Interface

This page details the `mojom::Renderer` interface, which is the primary interface used by the browser process to communicate with the renderer process.

## Overview

The `mojom::Renderer` interface is defined in `content/common/renderer.mojom`. It provides a set of methods that the browser process can call on the renderer process to control its behavior and receive information from it.

## Key Methods

-   `SetProcessState`: Sets the process state, including visibility and priority.
-   `SetIsCrossOriginIsolated`: Informs the renderer whether it is cross-origin isolated.
-   `SetIsIsolatedContext`: Informs the renderer whether it is an isolated context.
-   `SetIsWebSecurityDisabled`: Informs the renderer whether web security is disabled.
-   `SetIsLockedToSite`: Informs the renderer whether it is locked to a site.
-   `SetIsPdf`: Informs the renderer whether it is rendering a PDF.
-   `SetIsGuest`: Informs the renderer whether it is a guest process.
-   `SetIsSandboxed`: Informs the renderer whether it is sandboxed.
-   `SetIsOutOfProcess`: Informs the renderer whether it is running out-of-process.
-   `SetIsJitDisabled`: Informs the renderer whether JIT is disabled.
-   `SetAreV8OptimizationsDisabled`: Informs the renderer whether V8 optimizations are disabled.
-   `SetIsOriginAgentCluster`: Informs the renderer whether it is using the Origin-Agent-Cluster header.
-   `SetIsOriginAgentClusterRequestedByWeb`: Informs the renderer whether the Origin-Agent-Cluster header was requested by the web.
-   `SetIsOriginAgentClusterByDefault`: Informs the renderer whether the Origin-Agent-Cluster header is enabled by default.
-   `SetIsUnsubscribingFromOriginAgentCluster`: Informs the renderer whether it is unsubscribing from the Origin-Agent-Cluster.
